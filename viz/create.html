Optional overrides (auto-detected from PDF if left blank)
  • Map name ends with syllabus form + level, e.g. Chemistry 5070 — Cambridge O Level (2026–2028)
  • Works best with Cambridge PDFs using subtopics like 1.1 Solids, liquids and gases
  • Generation may take 10–30 seconds for large syllabuses
